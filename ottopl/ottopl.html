Otto Programming Language

Useless, full of bugs and meant just for fun


Keywords


Keyword Parameters Meaning
++ <comment> Marks a comment. Everything in a comment will not be executed. Attention: A comment must end with ++ again!
- Marks the end of a line. This character MUST be placed at the end of each line being executed. Comments do not need this!
_ <variable> Marks the following text up to the next space as a variable
$$ <key> Marks the following text up to the next space as a key that stores the index of the next line.
Otto <variable> <value> Creates a text variable that can be used to store text values
Uni <variable> <value> Creates a boolean variable that can be used to store boolean values
Guericke <variable> <value> Creates a numeric variable that can be used to store numeric values
Uni von <a> <b> Returns the sum of the numbers a & b. (a + b)
von Uni <a> <b> Returns the difference from the numbers a & b. (a - b)
Otto von <a> <b> Returns the product of the numbers a & b. (a * b)
von Otto <a> <b> Returns the quotient of the numbers a & b. (a / b)
Uni Magdeburg <a> <b> Returns the rest of a / b. (a % b)
Guericke von <a> <b> Returns true if a > b otherwise false
von Guericke <a> <b> Returns true if a < b otherwise false
von Magdeburg <a> Returns true if a is an odd number, otherwise false
Magdeburg von <a> Returns true if a is an even number, otherwise false
Magdeburg Guericke <a> <b> Returns true if a == b otherwise false
Guericke Magdeburg <a> <b> Returns true if a != b otherwise false
Otto Uni <a> <b> Returns true if a <= b otherwise false
Uni Otto <a> <b> Returns true if a >= b otherwise false
Otto Guericke <text> Prints out the given text into the console
Guericke Otto <variable> Retuns the value of the given variable
Magdeburg Uni <condition> <key> Jumps to the key, if the condition is fulfilled, otherwise this line is ignored.
Otto von Guericke Uni Magdeburg <condition> Exits the program if the condition is fulfilled, otherwise this line is ignored.

Examples

It doesn't matter if you set one or more tabs at the beginning of the line, because they will be removed anyway.

Otto Guericke Guericke Otto Hello, World!-
